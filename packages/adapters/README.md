# `@theneuralledger/adapters`

Host-neutral contracts for TNL Intelligence AI client integrations. The package maps the six
evidence-first research workflows to MCP tools, builds validated Tool 05 tasks, negotiates
capabilities, renders cited Markdown fallbacks, normalizes recovery guidance, and emits only
allowlisted telemetry fields.

It contains no host SDK, credentials, retrieval code, or research orchestration. Cursor and OpenAI
bundles are generated from the same catalog under `integrations/`.
