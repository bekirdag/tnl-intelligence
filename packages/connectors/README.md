# `@theneuralledger/connectors`

Shared action, trigger, lifecycle, pagination, and error contracts for the TNL n8n, Pipedream,
and Zapier integrations. Credentials and persistence are host-owned. Signed events are verified
against the exact raw body before JSON parsing and deduplicated by stable delivery/event IDs.

This package is read-only and does not execute trades or provide personalized investment advice.
