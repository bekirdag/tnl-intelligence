# TNL Intelligence 0.1.0

The first public TNL Intelligence tool-suite release provides a source-linked,
read-only intelligence and research layer for AI agents, automation systems,
developers, and quantitative researchers.

## Included

- TypeScript SDK, CLI/daemon, stdio and Streamable HTTP MCP server.
- Remote gateway, developer onboarding, webhook delivery, and event contracts.
- Evidence-first research orchestration, skills, MCP App, and responsive web UI.
- Cursor and OpenAI adapter bundles with shared cited-result rendering.
- n8n, Pipedream, and Zapier connectors backed by one normalized connector core.
- Python SDK with point-in-time datasets, event studies, and leakage controls.
- MCPB and client configuration artifacts, container image, OpenAPI snapshot,
  schemas, examples, operations guides, SBOM, provenance, and qualification
  evidence.

## Qualification

The release candidate passes contract, functional, security/privacy,
reliability, operations, artifact, and documentation gates. Six integrated
scenarios cover local and hosted MCP research, webhook automation, AI client
workflows, point-in-time quantitative use, and degraded dependency recovery.

The tools provide intelligence and research inputs only. They do not execute
trades, connect to brokers, provide trading-grade prices, or constitute financial
advice.
